# Briefly Website

Static landing page for Briefly.

## Deploy to Vercel

Deploy the `website` folder as the Vercel project root.

The AdMob file must be available at:

```text
https://briefly.vercel.app/app-ads.txt
```

Expected contents:

```text
google.com, pub-3452539673821547, DIRECT, f08c47fec0942fa0
```

After deployment, use `https://briefly.vercel.app` as the App Store Connect support or marketing URL, then run AdMob's app-ads.txt check again.
